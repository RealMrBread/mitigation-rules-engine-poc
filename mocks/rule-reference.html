All attic vents must be screened with ember-resistant mesh to prevent ember intrusion during a wildfire event. Standard screens are insufficient; 1/8" ember-resistant mesh is required.

Rule Logic

attic_vent_screens = "Ember Resistant"

Available Mitigations

Full

Install Ember-Rated Vents

Replace all vents with 1/8" ember-resistant mesh screens

Roof must meet fire resistance standards. Class A is required by default. However, if the property is in Wildfire Risk Category A (low risk), Class B is also acceptable.

Rule Logic

WHEN wildfire_risk_category = "A":

roof_type IN ["Class A", "Class B"]

DEFAULT:

roof_type = "Class A"

Available Mitigations

Full

Replace Roof with Class A Material

Install a Class A fire-resistant roof (concrete tile, metal, slate)

Windows must be a safe distance from surrounding vegetation. The required safe distance is computed from a base value modified by window type and vegetation type. Each vegetation item is evaluated independently.

Rule Logic

base = 30 ft

required = base x window_modifier x veg_modifier

Window: Single Pane=x3, Double Pane=x2, Tempered=x1

Vegetation: Tree=x1, Shrub=/2, Grass=/3

FAIL if vegetation[].distance < required

Available Mitigations

Full

Replace with Tempered Glass

Upgrade all windows to tempered glass (modifier becomes x1)

Bridge

Apply Protective Film (modifier: x0.8)

Apply protective window film to reduce heat transmission. Reduces required distance by 20%.

Bridge

Prune Nearby Trees (modifier: x0.5)

Prune trees within the flagged zone to reduce fire fuel load. Reduces required distance by 50%.

Rule Logic

home_to_home_distance ≥ 15.0 ft
